# Joy.js [![GH version](https://badge-me.herokuapp.com/api/gh/chrisenytc/joy.png)](http://joy.chrisenytc.com/for/gh/chrisenytc/joy)

> A awesome sails stack

## Usage

Install dependencies

`sudo npm install`

Build Project

`grunt`

Run Project

`sails lift`

## Contains

- * Oauth2
- * Oauth2 clients
- * Passport
- * Login
- * Signup
- * Logout
- * Profile
